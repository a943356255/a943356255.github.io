同步与异步

通俗点来讲就是调用方是否需要等待结果，如果需要等待结果，就是同步；如果不需要等待结果，就是异步。

异步实现的方法：

1、调用方创建一个子线程，在子线程中执行方法调用，这种调用我们称为异步调用；

2、方法实现的时候，创建一个新的线程执行主要逻辑，主线程直接 return，这种方法我们一般称为异步方法。
